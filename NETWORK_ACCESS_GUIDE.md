## 🛠️ 故障排除

### 如果无法访问，请检查：

1. **防火墙设置**
   ```bash
   # macOS
   sudo pfctl -f /etc/pf.conf
   
   # 或临时关闭防火墙测试
   sudo pfctl -d
   ```

2. **端口占用**
   ```bash
   lsof -i :3000
   ```

3. **网络连接**
   ```bash
   ping 192.168.1.207
   curl http://192.168.1.207:3000
   ```

4. **路由器设置**
   - 确保设备在同一子网
   - 检查AP隔离是否开启

### 常见问题解决

**Q: 移动设备无法访问**
A: 确保手机和电脑在同一WiFi网络，关闭VPN

**Q: 页面加载慢**
A: 首次访问需要编译，后续访问会很快

**Q: 管理后台无法访问**
A: 需要先登录，管理后台有身份验证
